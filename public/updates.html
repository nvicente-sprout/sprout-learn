Sprout Learn — What's New

June 2026 Version 1.2 → 1.5 Prepared for Exec Review

This covers improvements shipped to Sprout Learn over the past two months. Changes span the learner experience, admin visibility tools, and platform reliability — all live on production today.

4Bug Fixes

Learner Experience
🏆
New
Team Podium Leaderboard
The learner dashboard now shows a podium-style leaderboard scoped to their own team — 1st, 2nd, 3rd place displayed with medals. Encourages healthy competition within teams rather than across the whole company.
📦
Fix
SCORM Course Rendering
Uploaded SCORM packages were displaying raw HTML source code instead of rendering the course content. Fixed — SCORM courses now load and play correctly, with full progress tracking via the SCORM 1.2 bridge.
Admin Tools
🔍
New
Course Filter in Team Progress
Admins can now select a specific course in the Team Progress view to see only the members assigned to it, along with their individual completion status and score.
📊
New
Richer Member Progress Cards
Member cards in Team Progress now show three stats at a glance — Assigned courses, Completed courses, and Average Progress percentage — replacing the single number badge that was there before.
📥
New
Per-Course CSV Export
The Reports page now supports selective CSV exports. Admins can tick specific courses and download a report with Name, Team, Email, Course, Completion Status, and Score columns — one row per learner per course.
👤
New
Role Filter in Settings
The Settings member list can now be filtered to show Admins only or Learners only, making it easier to manage access across a growing user base.
🏷️
New
Publisher Name on Course Cards
Course cards now display the name of who uploaded the course. All newly created courses will show the uploader — historical courses without this data will show nothing rather than a placeholder.
Platform & Reliability
✨
New
Changelog Panel
A "What's New" panel is now accessible from the header (✨ icon) so any user can see a summary of recent updates without needing to ask — keeps the team informed without an email chain.
🔧
Fix
Category Filter Restored
The course category filter was silently broken in both the Admin and Learner course views — clicking a category showed all courses regardless. Root cause identified and fixed.
⚙️
Fix
Deployment Pipeline Restored
A configuration issue was preventing Vercel from building the serverless API functions (AI question generation). Resolved — all backend functions are deploying correctly again.
